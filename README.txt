# advportscanner.py

This is port scanner tool. Use it of your own risk. Thank you for using.

Before installing it you need to install termcolor module. After installing python using this command
pip install termcolor
After installing this pip your tool will work properly.


INSTALLATION STEPS:-)

git clone https://github.com/shuvopaul601/advportscanner.py.git

cd advportscanner.py

pip install termcolor

python3 main.py

Enjoy the tool ☺️☺️
